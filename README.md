# pylink
Just something to do to brush up on python


## Setup
```
sudo -H pip install pygame pyLint
sudo -H pip install pylint
```

## Test
```
pylint *.py && pytest -v && python pylink.py
```


## Run
```
python pylink.py
```

## Run (pylink3)
```
python3 pylink3.py
```
